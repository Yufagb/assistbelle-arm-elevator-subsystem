# Notas de seguridad

- Usar el robot solo en laboratorio o entorno controlado.
- Mantener las manos fuera del área de movimiento.
- Verificar límites articulares antes de trayectorias automáticas.
- Desconectar energía antes de cambiar cableado o piezas mecánicas.
- No exceder la carga ensayada.
- Detener operación si aparece vibración, ruido o calentamiento anormal.
- Mantener despejada el área de trabajo durante pruebas de manipulación.
