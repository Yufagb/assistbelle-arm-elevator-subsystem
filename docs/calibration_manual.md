# Manual de calibración

## Pasos mínimos

1. Definir cero mecánico de J1 a J5.
2. Verificar dirección de lectura de encoders.
3. Verificar respuesta de cada nodo CAN.
4. Verificar cámara y foco.
5. Ejecutar pruebas de rampa.
6. Ejecutar pruebas trapezoidales.
7. Guardar notas de calibración junto con los datos de validación.
