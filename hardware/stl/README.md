# Exportaciones STL

Carpeta canonica para archivos STL imprimibles en 3D.

Organizacion sugerida:

- joint_1_base/
- joint_2_shoulder/
- joint_3_elbow/
- joint_4_wrist/
- gripper/
- elevator_mounts/
- mounts/

Cada pieza debe incluir, si esta disponible, material, relleno, altura de capa y orientacion recomendada de impresion.
