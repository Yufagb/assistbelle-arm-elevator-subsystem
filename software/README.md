# Software

Esta carpeta es para notas de instalacion, herramientas auxiliares y archivos que no pertenecen directamente al workspace ROS 2.

El workspace principal de ROS 2 se mantiene en `ros2_ws/`.

Agregar aqui:

- notas de instalacion
- scripts auxiliares no ROS
- herramientas de graficacion
- scripts de procesamiento de datos
- notas de entorno de software

No copiar aqui nodos ROS 2 que ya existan en `ros2_ws/`.
