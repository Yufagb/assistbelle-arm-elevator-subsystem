# Electronica

Esta carpeta contiene la documentacion electrica necesaria para reproducir la electronica del robot.

## Subcarpetas

- schematics: esquematicos electricos y diagramas generales.
- wiring_diagrams: diagramas practicos de cableado para ensamblaje.
- pinout_tables: asignacion de pines para ESP32, drivers de motor, sensores y modulos CAN.
- power_distribution: fuente, convertidores, protecciones y notas de tierra.
- images: imagenes o fotos de referencia del cableado, controladores y potencia.

Antes de publicacion, cada nodo actuador debe tener una tabla clara de cableado y cada linea de alimentacion debe estar documentada.
